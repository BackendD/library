# کتابخونه
این پروژه صرفا جهت آشنایی بچه های جاوا با کتابخانه اسپرینگ و نحوه بکارگیری آن در یک پروژه REST میباشد.

در این پروژه از موارد زیر استفاده شده است:
* Java 11
* Spring Framework 5
* Postgresql
* REST
* MVC
* Gradle
* Git

هدف در این پروژه ترکیب موارد فوق و یادگیری کار با آنهاست.
